Detailed Description

Braille Alphabet with leds.

This class contains every letter of the alphabet in braille and requires the PCA9685 library. In the PCA9685 library you can find the code for the pwm pins, this class simply tells you which leds have to go on for each letter. NOTE: this is only done for PWM0, 1, 2, 3, 4 and 5. If you would want to use another PWM pin, you would have to assign it first.

void abcLed::Aled ( )

The A-Z led functions.

These are the braille alphabet functions that will show each corresponding letter in braille with leds. The leds will be on for 200 ms and then off again for 200 ms. For example, Aled turns on pwm(led)0, then waits 200 ms before turning it off again. The "write( 0x00, 0b00000001);" changes a setting in MODE1 of the chip, which will allow you to write to every register. This is added in the assignLetterToLed file to avoid having the same line of cover over and over again for each letter here. The ledSequence and AZledTest functions DO NOT require the assignLetterToLed file, it IS added there. To manually use one of the A-Zled functions WITHOUT the assignLetterToLed file, simply add the write line within that function. The assignLetterToLed file also starts with the allOff function, to avoid any errors with leds that might still be on. WITHOUT that file, it is recommended to call the allOff function manually, before using one (or more) of the A-Zleds below.

◆ allOff()

void abcLed::allOff ( )

This is the allOff function.

This will turn off every led, by calling the pwm-zero functions. (setting the register values to 0) Without this function, there's a chance that not all leds are off, which makes it much harder to see which led actually should be on. Only PWM 0,1,2,3,4 and 5 are done at the moment, since I only need those for my project.

◆ AZledTest()

void abcLed::AZledTest ( )

This is the AZledTest function.

This is an Alphabet Braille led testing sequence. This is simply to test if the leds turn on and off correctly for each letter. The "write( 0x00, 0b00000001);" changes a setting in MODE1 of the chip, which will allow you to write to every register. Since this is to test ALL the letters in this file, and it does NOT require the assignLetterToLed file, it IS added here. Then the allOff function will be called, followed by every letter of the alphabet. Currently this is done in a loop. At the end of the loop, the wait time is a bit longer (400 ms instead of the usual 200ms) to indicate a clearer start/end of the loop. If you DO NOT want this to loop, simply remove or comment the "while(1)" part with the corresponding {} signs.

◆ ledSequence()

void abcLed::ledSequence ( )

This is the ledSequence function.

This is a led testing sequence, unrelated to the alphabet. This is simply to test if the leds turn on and off. First all leds will be turned off, by using the allOff function, then each led will blink, one by one, from pwm0 till pwm 5. After that all leds will blink together and then turn off again. The "write( 0x00, 0b00000001);" changes a setting in MODE1 of the chip, which will allow you to write to every register. This is added in the assignLetterToLed file to avoid having the same line of cover over and over again for each letter here. Since this is the testing sequence for the leds, and it does NOT require the assignLetterToLed file, it IS added here.
