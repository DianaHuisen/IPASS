void assignLetterToMotor::letter ( char  ch)

Checks and assigns Alphabet letter to the correct abcMotor function.

This function checks for each character if it is a match, to see which (capital) letter it is. Then it calls the corresponding abcMotor function for it. If the input is invalid, a message will show up on your screen, this code only checks for A-Z.
